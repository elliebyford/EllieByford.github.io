    Autonomous Car Project

    In this project I designed and implemented autopilot software for autonomous car using AI. The project had 2 main objectives which were:

    1. To be able to follow a track using lane detection 2. To detect and respond to objects using obstacle detection e.g if an object is within a certain radius 30cm the car will automatically halt until the object is no longer near the car.

    • View Project

    In this project I used Raspberry Pi and the GoPiGo build kit along with the GoPiGo3 libraries. For implementation I used Python and I also utilised multiple libraries such as OpenCV for image processing techniques. I also implemented additional features which aided in training the car with Artificial Intelligence. An additional feature I implemented was the PlayStation Controller and I used the PyPS4 library to allow the controller to connect to the raspberry pi via bluetooth. Controlling the car with the controller and recording videos and images using the Raspberry Pi camera driving around the track before running the main program allowed the car to collect data of the track including its curve value and allowed it to be able to use pixel summation to calculate how many pixels where on each side of the track and this determined which direction the car would turn when coming across a curve or pattern in the track. Overall, I really enjoyed working on this project as autonomous cars, robotics and AI are all aspects of technology that really interest me and I'm glad I got the oppurtunity to work on this project.
